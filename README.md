# Invoice-Processing-Automation-Lean-Project-
This project focuses on improving a real-world invoicing process in a construction company using Lean principles and automation. 
The original process involved manual handling of subcontractor invoices and reporting.
Reduce manual work, minimize errors, and improve efficiency by automating data extraction and reporting.

Problems Identified
- Manual data entry from PDF invoices
- Time-consuming process
- High risk of human error
- Repetitive monthly workflow

Solution
- Automated invoice data extraction using Python
- Rule-based parsing of PDF files
- Automatic generation of monthly Excel report
- One-click process execution via simple interface

How It Works
1. Program reads all PDF invoices from a folder
2. Extracts key data (invoice number, amount, month)
3. Generates structured Excel report automatically
4. Process runs without manual input


Technologies
- Python
- PDF processing
- Excel automation
- Rule-based data extraction

My Role
- Process analysis (Lean methodology)
- Identifying inefficiencies
- Developing automation logic
- Testing and improving solution

Results
- Reduced manual work significantly
- Faster monthly reporting
- Lower error rate
- Scalable solution for future integration
